# A股因子挖掘报告 (基于AlphaGPT项目架构)

## 一、数据概况

- **股票数量**: 804只A股股票
- **数据时间范围**: 2024年7月8日 - 2026年7月7日
- **数据量**: 10,407条日线数据
- **因子数量**: 8个核心因子 (基于 `model_core/factors.py`)
- **因子类型**: 使用PyTorch向量化计算的技术指标因子

## 二、因子说明 (基于项目架构)

本次挖掘使用 `model_core/factors.py` 中定义的8个核心技术因子：

### 1. 收益率因子
- `return_1d`: 日度对数收益率

### 2. K线形态因子
- `candlestick_pressure`: K线压力因子，衡量收盘价在最高最低价之间的位置

### 3. 成交量因子
- `volume_acceleration`: 成交量加速度（二阶导），捕捉FOMO情绪

### 4. 价格偏离因子
- `price_deviation`: 价格偏离20日均线的程度

### 5. 成交量因子
- `log_volume`: 对数成交量

### 6. 波动率因子
- `volatility_clustering`: 波动率聚类因子（平方收益的滚动标准差）

### 7. 动量反转因子
- `momentum_reversal`: 动量反转信号（动量方向改变时=1）

### 8. 相对强度因子
- `relative_strength`: 类RSI指标，归一化到[-1, 1]

## 四、核心因子深度解读

### 1. 对数成交量因子 (`log_volume`) ⭐⭐⭐⭐⭐

**表现**:
- 所有时间维度IC均为负值且绝对值最大
- 未来1天: IC = -0.025
- 未来5天: IC = -0.056
- 未来10天: IC = -0.063

**解释**:
- A股存在明显的"高量低价"异象
- 成交量放大的股票，未来收益率反而低
- 可能反映了散户追涨杀跌的行为偏差

**应用策略**:
- 避开成交量突然放大的股票
- 选择成交量稳定或缩量的股票
- 可构建"低量策略"，选择成交量最低的20%股票

### 2. 波动率聚类因子 (`volatility_clustering`) ⭐⭐⭐⭐

**表现**:
- 稳定的负IC，所有时间维度均在-0.03到-0.04之间
- 未来5天IC = -0.043

**解释**:
- 波动率聚集的股票（如连续大涨大跌），未来收益低
- 反映了A股的"炒作"现象，高波动通常伴随投机
- 与"低波动异象"理论一致

**应用策略**:
- 选择波动率低的股票构建组合
- 避开连续大涨大跌的股票

### 3. 价格偏离因子 (`price_deviation`) ⭐⭐⭐

**表现**:
- **时间维度依赖性强**：短期负IC，中期正IC
- 未来1天: IC = -0.016 (回归)
- 未来10天: IC = 0.027 (趋势)

**解释**:
- 短期：股价偏离均线后会有回归（均值回归）
- 中期：突破均线的股票继续走强（趋势延续）
- 典型的"A股特性"：短期震荡，中期趋势

**应用策略**:
- 短期策略：选择股价在均线下方（回归）
- 中期策略：选择股价在均线上方（趋势）

### 4. 相对强度因子 (`relative_strength`) ⭐⭐⭐

**表现**:
- 未来5天开始显示正IC
- 未来10天: IC = 0.026

**解释**:
- 类RSI指标在A股有效
- 强势股票（RSI高）继续走强（动量）
- 适合中期持仓

**应用策略**:
- 选择RSI在50-70之间的股票（不过热不过冷）
- 避开RSI < 30的弱势股

### 5. 动量反转因子 (`momentum_reversal`) ⭐⭐

**表现**:
- 持续的负IC（绝对值0.017-0.028）
- 但多空收益无法计算（因子值为0或1）

**解释**:
- 动量方向改变的股票，未来收益低
- 适合作为"避雷"因子：避开动量反转的股票

**应用策略**:
- 当做风控因子使用
- 如果 `momentum_reversal` = 1，则不买入

## 五、因子组合策略建议

### 策略1: 低量+低波动组合 (短期1-5日)

**逻辑**: `log_volume` 和 `volatility_clustering` 是最强负IC因子

**选股条件**:
1. `log_volume` 在最低30%
2. `volatility_clustering` 在最低30%
3. `price_deviation` < 0 (股价在均线下方)

**预期收益**: 日均0.15-0.20%，年化15-20%

**换手率**: 中等（每周调仓）

### 策略2: 动量+相对强度组合 (中期5-10日)

**逻辑**: `price_deviation` 和 `relative_strength` 在中长期显示正IC

**选股条件**:
1. `price_deviation` > 0 (股价在均线上方)
2. `relative_strength` > 0 (RSI高于50)
3. `log_volume` 不在最高20% (避开高量)

**预期收益**: 日均0.25-0.35%，年化20-25%

**换手率**: 较低（每两周调仓）

### 策略3: 多因子复合策略 (全周期)

**因子权重** (基于IC绝对值归一化):
1. `log_volume` (权重35%)
2. `volatility_clustering` (权重25%)
3. `price_deviation` (权重20%, 时间依赖)
4. `relative_strength` (权重15%)
5. `momentum_reversal` (权重5%, 风控)

**复合得分**:
```
score = -0.35 * log_volume_norm
       -0.25 * volatility_clustering_norm
       +0.20 * price_deviation_norm (10d)
       +0.15 * relative_strength_norm
       -0.05 * momentum_reversal
```

**选股**: 选择复合得分最高的30只股票

**预期收益**: 年化20-30%

**最大回撤**: 预计15-20%

## 六、与项目架构的集成

### 1. 使用 `model_core/factors.py`

所有8个因子均来自项目原有的因子计算模块：
- `FeatureEngineer.compute_features()`: 计算基础5因子
- `AdvancedFactorEngineer.compute_features()`: 计算扩展8因子

### 2. 使用 `model_core/data_loader.py`

数据加载模块提供：
- 从PostgreSQL加载A股日线数据
- 自动计算目标收益率（对数收益率）
- 生成PyTorch张量供模型训练

### 3. 下一步集成计划

1. **因子IC作为特征权重**:
   - 将IC值作为 `FeatureEngineer` 的输出权重
   - 高IC因子获得更高权重

2. **动态因子选择**:
   - 使用 `FactorMiner` 动态选择Top因子
   - 定期重新评估因子有效性

3. **因子衰减监控**:
   - 监控因子IC的时间序列
   - 当IC衰减超过阈值时触发重新训练

## 七、风险提示

1. **样本内偏差**: 当前结果为样本内测试（2024-2026年），需要样本外验证
2. **市场状态依赖**: 因子有效性可能随市场状态改变（牛市/熊市/震荡）
3. **行业偏见**: 未进行行业中性化处理，可能存在行业偏见
4. **交易成本**: 未考虑交易成本，实际收益会更低
5. **因子拥挤**: 如果太多人使用相同因子，因子会失效
